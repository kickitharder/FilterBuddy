FilterBuddy

ASCOM driver for the FilterBuddy controller for the SBIG CFW-9 filter wheel.

The FilterBuddy controller is connected to the SBIG CFW-9's 9-pin female connector. The controller has an Arduino within it and it is connected to one of your PC's USB ports.

Your PC needs to have the appropriate driver for the Arduino. The official Arduino driver can be obtained by dowloading the Arduino IDE at the Arduino Webite.

Once everything is connected, FilterBuddy will be presented on a COM port. When selecting FilterBuddy in the ASCOM FilterWheel Chooser, click on the Properties button to select the COM port. Click on the "Check Connection" button to see if you are using the correct one.

Ensure you visit the FilterBuddy GitHub website regularly for the latest drivers and information.

Enjoy!
